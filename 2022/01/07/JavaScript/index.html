JavaScript 知识总结

一、JavaScript 简介

JavaScript 是一种基于原型、动态类型的解释型脚本语言，主要用于网页的交互行为控制。

  • 执行环境：浏览器、Node.js
  • 特点：弱类型、解释执行、单线程、异步编程

二、数据类型

原始类型（Primitive Types）

  • String
  • Number
  • Boolean
  • undefined
  • null
  • Symbol
  • BigInt

引用类型

  • Object
  • Array
  • Function
  • Date, RegExp, Map, Set

三、变量声明

  • var：函数作用域，存在变量提升
  • let：块级作用域，不提升
  • const：常量，块级作用域

四、运算符

  • 算术运算：+, -, *, /, %, **
  • 比较运算：==, ===, !=, !==, >, <, >=, <=
  • 逻辑运算：&&, ||, !
  • 位运算：&, |, ^, ~, <<, >>, >>>
  • 赋值运算：=, +=, -=, *=, /=, %=

五、流程控制

  • 条件语句：if, else if, else, switch
  • 循环语句：for, while, do...while
  • 跳转语句：break, continue, return

六、函数

定义函数的方式

参数默认值 & 剩余参数

七、对象与数组

对象定义与访问

数组方法

  • push, pop, shift, unshift
  • map, filter, reduce, forEach
  • includes, indexOf, slice, splice

八、作用域与闭包

  • 函数作用域 vs 块级作用域
  • 闭包：函数可以“记住”并访问定义时的作用域

九、异步编程

  • setTimeout, setInterval
  • 回调函数（Callback）
  • Promise
  • async/await

十、ES6+ 新特性

  • 解构赋值：const { name } = obj
  • 模板字符串：Hello, ${name}
  • 展开运算符：...arr, ...obj
  • 箭头函数：() => {}

十一、DOM 操作（浏览器端）

  • 获取元素：getElementById, querySelector
  • 修改内容/属性/样式：innerHTML, textContent, setAttribute, style
  • 添加事件监听：

十二、模块化

  • export, import

十三、常用工具函数

十四、调试与开发工具

  • console.log, console.error, console.table
  • 浏览器开发者工具：Elements, Console, Network, Sources
  • ESLint, Prettier, Babel
