前端基础知识总结

一、HTML：结构化页面

HTML（超文本标记语言）用于构建网页的结构。

常用标签

  • <!DOCTYPE html>：声明文档类型
  • <html>：HTML文档根元素
  • <head>：页面头部，包含<meta>、<title>、<link>等
  • <body>：页面主要内容区域
  • 常见结构标签：<header>、<nav>、<main>、<section>、<article>、<footer>
  • 文本：<h1>~`

  • 表单：<form>、<input>、<textarea>、<button>、<select>、<label>
  • 媒体：<img>、<video>、<audio>
  • 列表：<ul>、<ol>、<li>

二、CSS：美化页面

CSS（层叠样式表）用于控制网页的外观和布局。

选择器类型

  • 元素选择器：div, p
  • 类选择器：.class
  • ID选择器：#id
  • 组合选择器：div > p, ul li, a:hover

常用属性

  • 布局：display, position, margin, padding
  • 尺寸：width, height
  • 字体：font-size, font-family, color, line-height
  • 背景：background-color, background-image
  • 边框：border, border-radius
  • 动画：transition, animation

布局方式

  • Flexbox：适用于一维布局（主轴和交叉轴）
  • Grid：适用于二维布局（行和列）

三、JavaScript：页面交互逻辑

JavaScript 是网页的脚本语言，用于实现交互功能。

基本语法

  • 变量声明：let, const, var
  • 数据类型：Number, String, Boolean, Object, Array, null, undefined
  • 条件语句：if, switch
  • 循环语句：for, while, forEach
