计算机组成原理知识梳理

1. 计算机系统概述

1.1 冯·诺依曼结构

  • 存储程序控制
  • 指令和数据统一存储
  • 顺序执行（可中断）
  • 五大部件：
    • 运算器
    • 控制器
    • 存储器
    • 输入设备
    • 输出设备

1.2 计算机的层次结构

  • 数字逻辑层
  • 微程序控制层
  • 指令集结构层（ISA）
  • 操作系统层
  • 应用程序层

2. 数据的表示与运算

2.1 数制与编码

  • 常见进制：二进制、十进制、十六进制
  • 转换方法：除基取余，权展开法
  • 编码方式：
    • 原码
    • 反码
    • 补码（主流，便于加减法统一处理）
    • 移码（浮点表示中使用）
    • ASCII、Unicode、BCD

2.2 定点数与浮点数

  • 定点数：
    • 整数部分固定，适合简单快速运算
  • 浮点数（IEEE 754标准）：
    • 单精度（32位）：1位符号 + 8位阶码 + 23位尾数
    • 双精度（64位）：1位符号 + 11位阶码 + 52位尾数

2.3 运算方法

  • 加减法：补码加法，溢出判断
  • 乘法：
    • 移位加法器法（Booth算法）
  • 除法：
    • 恢复法、非恢复法

3. 存储系统

3.1 存储器分类

  • 主存储器（RAM、ROM）
  • 辅助存储器（硬盘、磁带等）
  • 高速缓存 Cache
  • 寄存器（CPU内部）

3.2 存储器层次结构

3.3 Cache

  • 局部性原理（时间局部性、空间局部性）
  • 映射方式：
    • 直接映射
    • 全相联映射
    • 组相联映射
  • 替换算法：LRU、FIFO、Random

3.4 主存地址映射

  • 连续分配
  • 分页存储管理
  • 段式管理

4. 指令系统

4.1 指令格式

  • 操作码 + 地址码
  • 常见长度：1、2、3地址指令

4.2 指令类型

  • 数据传送指令
  • 运算指令
  • 控制指令（跳转、分支）
  • 输入输出指令

4.3 寻址方式

  • 立即寻址
  • 直接寻址
  • 间接寻址
  • 寄存器寻址
  • 变址/基址寻址
  • 相对寻址

5. 中央处理器（CPU）

5.1 数据通路

  • 运算器（ALU）
  • 寄存器组（通用寄存器、专用寄存器）
  • 总线结构（单总线、双总线、三总线）

5.2 控制器

  • 控制单元类型：
    • 硬布线控制器
    • 微程序控制器

5.3 指令周期与执行流程

  • 取指（IF）、译码（ID）、执行（EX）、访存（MEM）、写回（WB）
  • CPI（Cycles Per Instruction）

5.4 流水线技术

  • 指令流水线的基本阶段
  • 冲突类型：
    • 结构冲突
    • 数据冲突
    • 控制冲突
  • 解决方法：冒险检测、转发、分支预测等

6. 输入输出系统

6.1 输入输出方式

  • 程序查询方式（轮询）
  • 中断方式（效率高）
  • DMA方式（直接内存访问）

6.2 I/O接口与总线

  • I/O地址空间映射
  • 总线结构：
    • 数据总线
    • 地址总线
    • 控制总线

7. 总线系统

7.1 总线类型

  • 单总线
  • 分级总线（系统总线 + I/O总线等）

7.2 总线仲裁

  • 集中式（优先级法、查询法）
  • 分布式（链式仲裁、令牌传递）

8. 常见性能指标

  • 主频（Clock Rate）
  • CPI（每条指令的平均周期数）
  • MIPS（百万条指令每秒）
  • 吞吐率（系统单位时间内完成任务数量）
  • 延迟（执行一条指令所需时间）
